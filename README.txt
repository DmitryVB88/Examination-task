Данная программа написана для сортировки массива. 
Оснувным условием является выбор элементов состоещих из 3-х и МЕНЕЕ символов. 
В программе задан ассив стринговых переменных. 
Выводим его не экрен по средством цикла for.
После чего, создаем новый массив, в который будем отсортировывать 
удволитворяющие вышеуказанному условию элементы.
Вводим интовую перемменную j, для того, чтобы выбранные элементы записывались подряд. 
Далее, выводим готовый результат на экран.
